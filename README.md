# Bibliotecas e Frameworks

* Android 4.2.2 (API 17)
* PhoneGap 2.5.0
* jQuery
* jQuery Mobile: framework para implementação da UI
* Backbone.js: framework para arquitetura e padrões de javascript
* qUnit: biblioteca de testes
